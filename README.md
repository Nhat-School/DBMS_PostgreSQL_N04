# DBMS_PostgreSQL_N04

A Database Management Systems (DBMS) project using PostgreSQL.

## Description

This repository contains coursework and projects related to Database Management Systems using PostgreSQL.

## Getting Started

### Prerequisites

- [PostgreSQL](https://www.postgresql.org/download/) installed on your system
- A PostgreSQL client (e.g., pgAdmin, psql CLI)

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Nhat-School/DBMS_PostgreSQL_N04.git
   cd DBMS_PostgreSQL_N04
   ```

2. Connect to your PostgreSQL server and create the database:
   ```sql
   CREATE DATABASE dbms_n04;
   ```

3. Run the SQL scripts to set up tables and data.

- **Database:** PostgreSQL
